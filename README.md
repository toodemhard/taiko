# Taiko
Remake of [Taiko no Tatsujin](https://en.wikipedia.org/wiki/Taiko_no_Tatsujin) from scratch in C++.

## Building
```
git clone https://github.com/toodemhard/taiko.git
cd taiko
mkdir build
cmake -S . -B build
cd build
cmake --build .        # --parallel for number of cpu threads
```
